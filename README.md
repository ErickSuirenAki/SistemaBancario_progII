##  Sistema Bancário em C

Este projeto consiste em um sistema bancário desenvolvido em linguagem C, com foco na manipulação de dados de clientes por meio de arquivos texto. O sistema simula operações básicas de uma conta poupança, permitindo o gerenciamento de contas de forma simples via terminal.

Os dados dos clientes são armazenados em um arquivo (`poupanca.txt`) e carregados para a memória durante a execução do programa, possibilitando operações como consulta, atualização e organização das informações.

O sistema conta com um menu interativo que permite realizar diversas funcionalidades, como:

- Leitura e carregamento dos dados do arquivo
- Exibição da lista de clientes
- Regravação dos dados atualizados no arquivo
- Busca de contas (utilizando busca exaustiva e busca indexada)
- Criação e ordenação de índices para otimização de buscas
- Realização de depósitos e saques com validação de saldo
- Inserção de novos clientes com geração automática de número de conta
- Geração de relatório ordenado por número da conta

Durante o desenvolvimento, foram aplicados conceitos importantes da programação estruturada, como uso de structs, manipulação de vetores, funções, ordenação (Bubble Sort), além de técnicas de busca e organização de dados.

O projeto foi desenvolvido na disciplina de Programação 2, no 2º período do curso de Ciência da Computação, com o objetivo de consolidar conhecimentos sobre manipulação de dados e arquivos em linguagem C.
